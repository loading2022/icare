# iCare 日照機器人
### 待開發功能
> 角色設定:character <br>
> 照片上傳:upload_img <br>
> 對話紀錄:record <br>
> 吃藥回診提醒:alert <br>
> 回報問題:report <br>
> 語音 style:style <br>

一律推到 branch 確定再 push 回 main

### Technologies Used
1. Azure Speech to text
2. OpenAI API
3. D-ID
4. Deepface & OpenCV

### Installation
1. Install required modules
```
pip install -r requirements.txt
```
2. Get API keys of Azure, OpenAI and D-ID. Then put them in .env file.
3. Execute `main.py`
